# Web Development Crash Course: From HTML to WordPress Plugins

[![Course Status](https://img.shields.io/badge/status-complete-brightgreen.svg)]()
[![License](https://img.shields.io/badge/license-MIT-blue.svg)]()

A comprehensive 6-week journey from front-end fundamentals to WordPress plugin development. This course takes you from absolute beginner to building custom WordPress plugins.

## 📋 Course Overview

| Week | Topic | Skills Gained |
|------|-------|----------------|
| 1 | Web Foundations & HTML | Semantic HTML, web architecture |
| 2 | CSS Styling | Responsive design, Flexbox |
| 3 | JavaScript Interactivity | DOM manipulation, events |
| 4 | Server-Side PHP | Form handling, MySQL basics |
| 5 | WordPress Themes | Template hierarchy, The Loop |
| 6 | WordPress Plugins | Hooks, shortcodes, Options API |

## 🎯 Who This Course Is For

- **Absolute beginners** to web development
- **Front-end developers** wanting to learn back-end skills
- **Designers** transitioning to development
- **WordPress users** wanting to customize their sites

## 📚 Course Structure

### Week 1: Web Foundations & HTML
- Client-server model, HTTP/HTTPS, DNS
- HTML5 semantic elements
- Multi-page website structure

**Project:** Personal profile website (About Me, My Hobbies, Contact)

### Week 2: CSS Styling
- Box model, Flexbox layouts
- Responsive design principles
- Media queries

**Project:** Style the HTML site with responsive navigation

### Week 3: JavaScript Interactivity
- DOM manipulation and events
- Variables, functions, control flow
- Event handlers

**Project:** Dark mode toggle, image slider, or form validation

### Week 4: Server-Side PHP
- PHP syntax and superglobals
- Form processing and validation
- MySQL database integration

**Project:** Contact form with PHP processing (optional database storage)

### Week 5: WordPress Themes
- WordPress architecture and dashboard
- Converting static HTML to WordPress
- The WordPress Loop

**Project:** Convert static site into a WordPress theme

### Week 6: WordPress Plugins
- Plugin file structure and headers
- Actions vs. Filters
- Shortcode API
- Options API

**Project:** "Custom Welcome Message" plugin with admin settings

## 🛠 Prerequisites

- Basic computer literacy
- Text editor (VS Code recommended)
- Web browser (Chrome/Firefox)
- Local server environment for weeks 4-6 (Local by Flywheel or XAMPP)
